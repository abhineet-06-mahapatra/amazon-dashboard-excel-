# amazon-dashboard-excel-

**🔍 Overview**
Created an interactive Amazon Sales Dashboard in Microsoft Excel. The dashboard provides business insights using Pivot Tables, Pivot Charts, KPI Cards, and Slicers for dynamic filtering and analysis.

**✨ Features**
📈 Monthly Sales Trend Analysis
📦 Sales by Category
🌍 Country-wise Sales Analysis
🛒 Top 10 Products by Revenue
💳 Payment Method Analysis
🚚 Order Status Distribution
🎯 Interactive Slicers for Filtering
📌 KPI Cards for Revenue, Orders,Avg order, Quantity Sold & Delivery %
**🛠 Tools & Skills Used**
📊 Microsoft Excel
📌 Pivot Tables
📉 Pivot Charts
🎨 Dashboard Designing
📍 Slicers & Interactive Filters
📑 Data Cleaning & Formatting
📈 Business Data Analysis
**💡 Key Insights**
Electronics category generated the highest revenue.
Delivered orders contributed the majority of total sales.
United States and India showed strong sales performance.
Credit Card and UPI were among the most preferred payment methods.
Top-selling products contributed significantly to overall revenue.
⭐ This project demonstrates data visualization, dashboard development, and business analysis skills using Microsoft Excel.
